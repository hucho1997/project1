### DB 현황
- PokeAPI: 145 테이블 (포켓몬, 기술, 타입, 특성, 진화, 아이템, 맵, encounter 등)
- gym_leaders → pokemongymleaders: 관장/사천왕/챔피언 파티 (레벨+기술배치)
- encounter 데이터: 7세대(USUM)까지만 유효. 8~9세대+Legends는 0행
- encounter method 포함: walk, surf, 낚시, gift, gift-egg, only-one, roaming, npc-trade, pokeflute, squirt-bottle 등
- 한국어 맵 이름: 3/6/7세대만 있음. 1/2/4/5세대는 영어 표시 (나중에 보완)
- 아이템 획득 위치, 필드 아이템, 맵 연결 관계 데이터 없음

### 도감 도우미 CLI 현재 상태
구현 완료:
- 세대/게임 선택 (1~7세대, 복수 선택)
- 전국도감/지방도감 선택
- 포켓몬 목록 (번호순, 포획 가능 O/- 표시, 일괄 조회로 속도 개선)
- 상세: 이름(한/영), 타입, 특성(숨겨진 포함), 종족값
- 진화: 한 줄 표기, 현재 포켓몬 볼드, 조건 표시
- 포획 위치: 선택한 버전만 표시 (version_group 누출 수정 완료)
- 레벨업 기술 + TM/HM (펼쳐보기)
- 맵별 출현 포켓몬 탐색 (m 키): 지방→맵 선택→버전별/방법별 출현 목록

### 도감 도우미 설계 [합의]
- CLI 먼저 → 로직 검증 후 웹 전환 (로직/출력 분리 완료)
- 진화 표기: 직선=한 줄, 분기=/ 구분, 현재 포켓몬 볼드
- 기술: 레벨업 기본 표시, TM/HM은 버튼으로 펼침
- 한국어 맵 이름 보완은 후순위
- 특정 버전에서만 나오는 포켓몬 필터 기능 (미구현)
