### 도감 완성 도우미 설계 [합의]

#### 사용자 경험 (설계 근거)
- 현재 4세대 전국도감 목표, HG 기반으로 성도도감 진행 중
- SS/D/P/Pt 모두 클리어 상태
- 박스에 전국도감 번호순 정리 → 빈 칸을 인게임도감/웹에서 확인하며 채움
- 핵심 니즈: 다른 버전 켤 때 "이번에 뭘 잡아와야 하는지" 한눈에 보기
- 교환진화는 몰아서 처리

#### 첫 화면: 세대+게임 선택 (한 화면)
```
목표: ◉ 지방도감  ○ 전국도감

4세대
├── 신오 [Diamond ☐] [Pearl ☐] [Platinum ☐]
└── 성도 [HeartGold ☐] [SoulSilver ☐]
```
- 세대 체크 → 전국도감 기준 (가장 높은 세대)
- 지방 체크 → 마이너체인지 버전(Pt/Crystal/Emerald) 지방도감 기준
- 복수 세대 선택 가능 (3+4세대 등)

#### 포켓몬 상세 정보 표시
- 이름, 타입, 특성
- 종족값
- 진화 정보: 한 줄 표기, 현재 포켓몬 볼드
  - 직선: 피츄 → **피카츄** → 라이츄(천둥의돌)
  - 분기: **이브이** → 샤미드(물의돌) / 쥬피썬더(천둥의돌) / ...
  - 클릭 시 해당 포켓몬으로 이동 (웹 전환 시)
- 포획 위치: 버전별 맵 + 방법 (레벨 제외)
- 레벨업 습득 기술 (기본 표시)
- 기술머신/비전머신 (버튼으로 펼쳐보기)
- 특정 버전에서만 출현하는 포켓몬 필터 기능

#### 개발 방침
- CLI 먼저 → 로직 검증 후 웹 전환
- 로직과 출력 분리 (전환 시 출력부만 교체)
- 목록은 전국도감 번호순
